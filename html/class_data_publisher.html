◆ addSecondaryTransmissionMode()

void DataPublisher::addSecondaryTransmissionMode ( TransmissionBase *  instance)

Add a secondary Transmission mode object.

Low priority secondary destination. This destination will only be published to if successfull. When the destination fails to accept the data it will not be resend. Only the primary will be used for this. Sending from local storage will be aborted on FAILED_TO_SEND
